# Alchemy for D&D & GURPS

A web application for managing custom homebrew alchemy rules for tabletop role-playing games like GURPS and D&D. Built with PHP, it provides an interactive interface for players and game masters to track ingredients, combine items at a brewing station, roll virtual dice, and log their creations.

## Features

- **Interactive Dashboard:** Dynamic animations that release bubbles when hovering over the "Start Brewing" button.
- **Ingredient Management:** Comprehensive data tables to easily view, filter, and select available alchemical components.
- **Brewing Station:** A dedicated interface to combine selected recipes and prepare your concoctions.
- **Integrated Dice Rolling:** Automatically throws a virtual die after a recipe is brewed to determine the success or potency of the potion.
- **Brewing Journal:** A persistent log accessible via the bottom-right menu to track past alchemical creations and results.
